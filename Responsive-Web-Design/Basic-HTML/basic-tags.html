Basic Tags

Headings

Provide hierarchical semantics to html documents

<h1>heading 1</h1>
<h2>heading 2</h2>
<h3>heading 3</h3>
<h4>heading 4</h4>
<h5>heading 5</h5>
<h6>heading 6</h6>

Paragraph

Paragraphs are placed inside <p>paragraph text</p> tags

Comments

Comments start with <!-- and end with -->

<!-- this is a comment -->
<!-- this is a comment
     over multiple lines -->

Structure

html 5 introduced new tags that give semantic structure to documents

<main></main>
<header></header>
<footer></footer>
<nav>for navigation</nav>
<article></article>
<section></section>

and more.

Images

to add images use <img> tag. For example
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="a cute upside-down kitten">
Note: the tag is self closing; the src attribute tells the browser where to find the image; and, the alt attribute is used to specify the alternate text for an image. It is useful when an image cannot be displayed and for screen readers.

Links

Link to other pages with the anchor tag:
<a href="https://www.freecatphotoapp.com">freeCodeCamp</a>
the href attribute provides the address to link to. It can also be used to link internally:
<span id="destination">Jump to here</span><a href="#destination">Jump from here</a>
Or, a hash on its own will create a dead link:
<a href="#">Dead link</a>
You can nest other elements, like images, inside anchor tags to make the the nested element into a hyperlink
<a href="#"><img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="a cute upside-down kitten"></a>

Lists

Unordered lists

Create bullet lists. Can be nested to give sub bullets

<ul>
   <li>item 1</li>
   <li>item 2</li>
      <ul>
         <li>item 3a</li>
         <li>item 3b</li>
      </ul>
   <li>item 4</li>
</ul>

Ordered lists

Create numbered lists.

<ol>
   <li>item 1</li>
   <li>item 2</li>
   <li>item 3</li>
</ol>
